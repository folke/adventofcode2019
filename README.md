# Usage

<!-- usage -->

```sh-session
$ npm install -g adventofcode
$ adventofcode COMMAND
running command...
$ adventofcode (-v|--version|version)
adventofcode/0.0.0 darwin-x64 node-v12.11.1
$ adventofcode --help [COMMAND]
USAGE
  $ adventofcode COMMAND
...
```
